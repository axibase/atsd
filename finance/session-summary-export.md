# Statistics

## Description

Retrieves session summary statistics for the instrument in JSON format.

## Request

| **Method** | **Path** |
|:---|:---|
| `POST` | `/api/v1/trade-session-summary/get` |

### Payload Parameters

| **Name** | **Description** |
|:---|:---|
| `entity` | **[Required]** Set to `<symbol>_[<class_code>]`. |

```json
{
  "entity": "gazp_[tqbr]"
}
```

## Response

The payload contains statistics at the end of each session and auction stage, identified by `key`.
