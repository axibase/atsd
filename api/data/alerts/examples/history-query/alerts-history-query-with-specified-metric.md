# History Query with Specified Metric

## Description

## Request

### URI

```elm
POST /api/v1/alerts/history/query
```

### Payload

```json
[
  {
    "metric": "disk_used_percent",
    "endDate": "2016-06-25T18:00:00Z",
    "interval": {
      "count": 1,
      "unit": "DAY"
    },
    "limit": 2
  }
]
```
