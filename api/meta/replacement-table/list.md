# Replacement Table: List

## Description

Retrieves a list of replacement table names.

## Request

| **Method** | **Path** | **Content-Type Header** |
|:---|:---|:---|
| GET | `/api/v1/replacement-tables/csv` | `text/csv` |
| GET | `/api/v1/replacement-tables/json` | `application/json` |

### Query Parameters

None
